Project Overview
Less Ramp Time Due to Consistent and intuitive UX

Logistics is a business area that includes many different structures and all these structures should work in a coordinated way and should not be disrupted.

Huns’ modern and well-designed user interfaces help to streamline your operations. In addition, it provides powerful integrations with shipping lines and accounting platforms.

How I Started to Design:
  • User analysis is done before starting to design the project. (age, education level, logistics experience, etc.)
  • Users were interviewed about product features and their expectations were learned in general.
  • After discussing features with the Product Manager and Backend Engineer team, I created a design sprint.
  • I planned and drew my designs in Figma. After completing the necessary tests and scenarios. I started to the coding stage of the designs. I delivered the product after testing again in the back-end and integration process.
