Project Overview
Search and find and select events that suit you is now easier than ever.

You can simply browse and search for events if you want to follow it The latest application about events that are currently open or upcoming, and you can easily choose events such as concerts, sports and various other events.

How I Started to Design:
  • First, I created a basic design system and easily built the basic structures.
  • I did Low Fidelity Wireframe and Medium Fidelity Wireframe design work. In this way, I quickly prepared the skeleton of the application and was able to test its navigation.
  • I made the mobile design suitable for the smartwatch.
  • Finally, the High Fidelity Wireframe was prepared. I tested the necessary scenarios and completed my design.
